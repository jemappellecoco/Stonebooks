📋 課程介紹

🎯

學習目標

  • • 強化初級英檢聽力和閱讀能力需求
  • • 訓練不需字字聽懂也能解題之技巧
  • • 必考題型整理，輕鬆回答刁鑽難題
👥

適合對象

國小五年級以上，準備報考英檢初級的學生。

建議英文程度：

已學過基本句型與常用單字，能理解簡單英文指令。

能力證明

🎧

聽力課程

透過初級英檢常見的聽力題型與技巧訓練，結合單字與句型應用，幫助學生熟悉題型與答題節奏。

✓ 看圖辨義訓練
✓ 問答句型練習
✓ 情境對話理解
📖

閱讀課程

搭配 11 種核心閱讀題型設計，提升閱讀理解力與應考信心，同時拓展主題視野與知識廣度。

✓ 多元主題文章
✓ 實用文體練習
✓ 閱讀策略指導

📅 課程規劃

堂數 聽力 閱讀
第1堂 看圖辨義 航空旅遊
第2堂 Wh問句 學習的未來
第3堂 Yes-No問句 古代歷史
第4堂 陳述句 健康
第5堂 大意題／活動題 環境
第6堂 位置題／職業題 科技
第7堂 理由／方法／推論題 廣告
第8堂 時間／狀況／圖表整合題 公告類文章
第9堂 廣告／公告 指示說明文章
第10堂 語音留言／電話留言 信件／電子郵件
第11-15堂 英檢模擬聽力與閱讀整合練習

📚 教材使用

📘 GEPT專用教材

本課程使用專為全民英檢設計的教材，內容涵蓋：

  • • 聽力與閱讀題型分析與解題技巧
  • • 初級英檢常用單字與句型整理
  • • 模擬試題與歷年試題精選
  • • 自學練習與複習資料

教材費用已包含在課程費用中，學生無需額外購買。
